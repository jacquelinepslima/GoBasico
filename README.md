## GoBasico

This repository contains practical exercises and code examples for learning the Go (Golang) programming language. The goal is to provide a solid understanding of Go fundamentals, helping learners move toward more advanced projects.

### 🚀 Getting Started

To run the examples locally, follow these steps:
#### 1. Clone the repository:
```bash
git clone https://github.com/jacquelinepslima/GoBasico.git
cd GoBasico
```

#### 2. Run a Go file:
```bash
go run <filename>.go
```
Example:
```bash
go run HelloWorld.go
```

### 📂 Repository Structure

The repository is organized as follows:

- HelloWorld.go – Your first Go program.
- Arrays.go – Working with arrays.
- Slices.go – Using slices.
- Maps.go – Using maps.
- Functions.go – Defining and using functions.
- IfStatement.go – Conditional statements with if.
- SwitchStatement.go – Conditional statements with switch.
- ForLoop.go – Loops with for.
- WhileLoop.go – Loops with while pattern.
- Pointers.go – Working with pointers.
- StructsAndInterfaces.go – Defining structs and interfaces.
- Goroutines.go – Introduction to goroutines.
- Defer.go – Using defer.
- Panic.go – Error handling with panic.
- Recover.go – Recovering from errors using recover.
- TypeConversions.go – Converting between data types.
- FinalProject.go – Final project integrating learned concepts.
